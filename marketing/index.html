Questions you're probably about to ask

Does AgentMonitor send my data anywhere?

No. Everything stays on your laptop. AgentMonitor runs a local server on localhost:8765 and writes to a local SQLite database. There is no cloud, no signup, no telemetry. We literally cannot see your data — by design.

Which AI agents does it support?

Anything that calls OpenAI, Anthropic, or a local model via Ollama — plus framework-level integrations for LangChain, AutoGen, Smolagents, Cursor's agent mode, and Claude Code. If you're using something else, the generic adapter takes 3 lines.

Is it free?

Yes. Core recording, replay and run history are free forever. Advanced features like Code Scan, Scanner Obs and sandbox Detonations are marked PRO in the sidebar — they're free during beta, and will be paid afterwards. We'll never paywall the core flight-recorder.

Do I have to change my code?

For most users, no — once installed, AgentMonitor auto-detects supported runtimes and starts recording. For custom agents, wrap your run in a with monitor(agent="my-agent"): block. That's it.

How is this different from LangSmith / Helicone / Phoenix?

Those are cloud-first observability platforms for prompt engineers. AgentMonitor is a local-first flight recorder for developers who run AI agents on their own machine and want to know exactly what those agents did. Different audience, different shape.

What's on the roadmap?

Today: record + replay + cost. Next: agent-level diff-and-revert (the "rewind" button stops being metaphorical). After that: a kill-switch that pauses a running agent the moment it tries something out of character. See our GitHub.
