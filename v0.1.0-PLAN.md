> **GitHub repo:** <https://github.com/cogniguardai/agentmonitor>.

## 1. Goal & success criteria

A user on a clean machine can:

```bash
pip install cogniguardai
agentmonitor                # opens the dashboard at http://localhost:8765
```

…and get a working AgentMonitor dashboard in under 60 seconds, with:

- The full FastAPI server + web UI bundled.
- SQLite DB auto-initialised at the OS-appropriate location.
- No heavyweight ML deps required (PyTorch / transformers move to opt-in
  `[ml]` extras).
- The 5 LLM adapters (openai, anthropic, langchain, ollama, smolagents,
  autogen) importable without errors.
- Mythos-specific code (NLA, code-scan, interp probes, sandbox detonations)
  either deferred to v0.2.0 or behind optional extras.

**Definition of done:** `pip install cogniguardai` in a fresh venv on
Windows 11 + Python 3.11 succeeds in <30 sec, downloads <50 MB, and
`agentmonitor` launches the dashboard window with no errors in
`%TEMP%\AgentMonitor.log`.

## 5. Refactor list (concrete code changes)

### 5.1 `agent_monitor/api.py` (Refactor 2.1)

Replace top-level imports:

```python
# old (lines 38-40)
from agent_monitor import db, interp_bridge, memory, nla_client, nla_cache, nla_worker
from agent_monitor import browser as browser_mod
from agent_monitor import code_scan as code_scan_mod
```

With a feature-availability pattern:

```python
from agent_monitor import db

# Optional MVP-deferred features. Endpoints that need them return 503
# when the import fails, but the rest of the dashboard remains usable.
def _try_import(name: str):
    try:
        import importlib
        return importlib.import_module(name)
    except Exception:
        return None

interp_bridge = _try_import("agent_monitor.interp_bridge")
memory        = _try_import("agent_monitor.memory")
nla_client    = _try_import("agent_monitor.nla_client")
nla_cache     = _try_import("agent_monitor.nla_cache")
nla_worker    = _try_import("agent_monitor.nla_worker")
browser_mod   = _try_import("agent_monitor.browser")
code_scan_mod = _try_import("agent_monitor.code_scan")
```

Every endpoint that uses an optional module gets a guard:

```python
@app.get("/api/interp/status")
def interp_status():
    if interp_bridge is None:
        raise HTTPException(503, "interp probe feature not installed; "
                                 "see https://cogniguardai.com/docs/extras")
    return interp_bridge.status()
```

### 5.2 `agent_monitor/runner.py` (Refactor 2.2)

Change line 28:

```python
# old
from agent_monitor import db, interp_bridge, memory
```

To:

```python
from agent_monitor import db
try:
    from agent_monitor import interp_bridge
except Exception:
    interp_bridge = None
try:
    from agent_monitor import memory
except Exception:
    memory = None
```

And in `_score_and_record()` and the memory-write helper, gate calls:

```python
if interp_bridge is None:
    return  # interp probes not installed; skip scoring
```

Drop the bottom of `runner.py` containing Mythos-specific helpers
(`run_customer_support_tickets`, `run_sop_processor`, etc.) or move
them to a new `examples/mythos_runners.py` outside the package.

### 5.3 `agent_monitor/wrappers.py` (Refactor 2.3)

**Remove from the wheel entirely.** It imports `core.engine.RecurrentDepthEngine`,
`core.kairos.KairosController`, and the `automations/` package — all
Mythos-specific. Move the file to `examples/` or a sibling directory
outside `agent_monitor/`, or delete it.

If we want to keep the file in the repo for Mythos's internal use, add
to `MANIFEST.in` / `pyproject.toml`'s `tool.setuptools.exclude-package-data`.

## 8. Pre-publish test checklist

1. **Fresh venv smoke**

   ```bash
   python -m venv /tmp/cog-test
   /tmp/cog-test/bin/python -m pip install dist/cogniguardai-0.1.0-py3-none-any.whl
   /tmp/cog-test/bin/agentmonitor --no-window &
   sleep 5
   curl -s http://127.0.0.1:8765/api/status | jq .
   ```

2. **No accidental Mythos imports**

   ```bash
   /tmp/cog-test/bin/python -c "import agent_monitor; import agent_monitor.api; import agent_monitor.runner"
   # should succeed with no ImportError
   ```

3. **Optional features return 503, don't crash**

   ```bash
   curl -s -o /dev/null -w "%{http_code}\n" http://127.0.0.1:8765/api/interp/status
   # expect: 503
   curl -s -o /dev/null -w "%{http_code}\n" http://127.0.0.1:8765/api/status
   # expect: 200
   ```

4. **Wheel size sanity**

   ```bash
   ls -lh dist/cogniguardai-0.1.0-py3-none-any.whl
   # target: <500 KB
   ```

5. **Install in <30 sec** (no torch, no transformers, no playwright):

   ```bash
   time pip install cogniguardai
   # target: <30s on broadband
   ```

## 11. Risks & known issues

| Risk | Severity | Mitigation |
|---|---|---|
| `api.py` refactor breaks endpoints we don't notice | Medium | Smoke test all `/api/*` endpoints (§8.3) before publish. |
| `pywebview` install fails on Linux (needs system deps) | Medium | Move pywebview to `[desktop]` extra; fall back to opening browser via `webbrowser.open()`. |
| Pinned dep range too narrow → conflicts with users' env | Low | Use minimum-version pins only. |
| User on Python 3.9 can't install | Low | `requires-python = ">=3.10"` — clear error. Document. |
| First-run SQLite migration fails on Windows path with spaces | Low | `db.init_db()` uses `Path` objects; already handled. |
| Marketing screenshots reference Mythos features (NLA, code-scan) that 503 in v0.1.0 | High | Either: (a) take new screenshots showing only MVP features, or (b) note "v0.2 features" caveats in dashboard UI. **TODO before launch.** |
